# ELEVATE CREATIVE — אפיון האתר (Spec)

מסמך אחד שמרכז את כל מה שסוכם והוחלט עד עכשיו: המוצר, ההשראה, השפה העיצובית,
המבנה, האנימציות, הארכיטקטורה הטכנית, וההרצה/דיפלוי. נכון לגרסה האחרונה.

---

## 1. המוצר

**Elevate Creative** — סוכנות דיגיטל וקריאייטיב. האתר הזה הוא **כרטיס הביקור הדגל**
של הסוכנות עצמה, ולכן הוא חייב להיות הדבר הכי מרשים שאפשר ("אפקט וואו").

- **קהל:** עסקים שרוצים אתר/חוויה דיגיטלית פרימיום.
- **המסר:** "אנחנו לא רק בונים אתרים — אנחנו מזקקים את הזהות של כל עסק ותופרים לו
  חוויה דיגיטלית שמרימה אותו מעל המתחרים."
- **שפה:** עברית, **RTL** מלא. מונחים לועזיים/מותג נשארים באנגלית (ELEVATE, תוויות מטא).

## 2. ההשראה וההחלטה המרכזית

האתר בנוי בהשראת **off-brand.com (Off+Brand)** — אבל מותאם למותג ולוגו של Elevate.
מה ש"גנבנו" מהם והתאמנו:

- שדה צבע **מינימליסטי** (כהה‑חם / בהיר‑חם), הרבה אוויר, איפוק.
- **טיפוגרפיה ענקית במשקל בינוני** (לא Bold כבד), בגוון אפור‑בהיר.
- **אלמנט מרכזי "חי"** שנודד דרך הסקשנים, **והטקסט הענק עובר מעליו עם mix-blend** —
  האותיות מחליפות גוון איפה שהן חוצות אותו. **זה אפקט החתימה.**
- כפתורי pill, לוגוטייפ, כפתור תפריט צף, גריד עבודות (bento), קווי מסלול, marquee.
- **השתלטות גרדיאנט** מלא‑מסך בתחתית.

> **ההבדל מ‑Off+Brand:** האלמנט המרכזי שלנו הוא **החץ הכחול** (הלוגו של Elevate),
> לא כדור צבעוני. החץ צבעוני/כחול כדי שאפקט ה‑blend יעבוד, אבל הזהות נשארת כחולה.

## 4. האלמנט המרכזי — "החץ החי"

ההחלטה הסופית אחרי כמה איטרציות:

- **דו‑מימדי** (לא תלת‑מימד / לא WebGL). נקי, לא "נוזלי‑רקוב".
- **גרדיאנט נוזלי שזורם** — כמה כתמי צבע מטושטשים (blobs) שנעים ומתערבבים
  בתוך **צורת החץ** (clip-path), בדיוק כמו פני הכדור של Off+Brand. "חי" באמת.
- **הדגשה מבריקה** (highlight לבן) שעוקבת אחרי העכבר + **הטיה תלת‑ממדית עדינה** לכיוון
  הסמן + **"נשימה"** מתמשכת (scale) + **נקודת לוויין** שמקיפה.
- **נודד בגלילה:** מרכז → ימין (מניפסט) → שמאל (עבודות) → חזרה למרכז (קשר).
- **נעלם** (fade) בחלק הגרדיאנט בתחתית.
- צבעוני (תכלת→כחול→עמוק) כדי שהטקסט מעליו יחליף גוון.

צורת החץ (SVG path, viewBox `0 0 120 132`):
```
M60 5 C68 27 90 78 114 128 C95 104 70 84 60 72 C50 84 25 104 6 128 C30 78 52 27 60 5Z
```
ל‑clip של ה‑blobs משתמשים באותו path מנורמל (0–1, `clipPathUnits="objectBoundingBox"`).
החץ משמש גם כ‑**"A" בלוגו** `ELEV⌃TE` וב‑favicon ובמסך הטעינה.

## 5. אפקט הטקסט (החתימה)

הכותרות הענקיות (`.blend`) משתמשות ב‑**`mix-blend-mode: difference`** עם דיו בהיר
(`--blend-ink: #dcdcdc`) **בשני המצבים**. הטריק:
- מעל רקע **בהיר** → ה‑difference מרנדר את הטקסט **כהה** (קריא).
- מעל רקע **כהה** → מרנדר **בהיר** (קריא).
- מעל **החץ הצבעוני** → האותיות **מחליפות גוון** (כתום/ירוק/סגול) בשני המצבים.

> כדי שזה יעבוד, ה‑canvas/אלמנט המרכזי חייב לשבת ב‑`z-index:-1` ו‑`main` חייב להישאר
> בלי stacking context (בלי `z-index`/`transform`), אחרת ה‑blend לא "מגיע" לחץ.

## 6. מבנה האתר (עמוד אחד, RTL)

1. **ניווט (קבוע):** לוגוטייפ `ELEV⌃TE Creative` (החץ במקום ה‑A) · קישורים (מניפסט, עבודות) ·
   מתג כהה/בהיר · CTA "צרו קשר". כפתור תפריט צף בתחתית.
2. **Hero:** החץ החי במרכז + 3 מילים ענקיות סביבו — **"חוויות / דיגיטליות / שמרימות."** +
   רמז גלילה.
3. **מניפסט / אודות (`#about`):** משפט ענק — *"ברגש, אסטרטגיה ועיצוב — אנחנו דוחפים את
   גבולות היצירה הדיגיטלית."* + תווית `(01) — WHO WE ARE` + פסקת lead (תיאור העסק).
4. **עבודות נבחרות (`#work`):** תווית `(02) — FEATURED WORK` + pill "כל העבודות" +
   **גריד bento** של 4 כרטיסי דמו (מדיית גרדיאנט, כותרת, תגיות) + הערת "עבודות לדוגמה".
5. **רצועת CTA (marquee):** "כאן השונה הוא הסטנדרט · בואו נעלה שלב" שרץ.
6. **צור קשר (`#contact`):** תווית `(03) — LET'S TALK` + כותרת ענק "מוכנים לעלות שלב?" +
   תת‑כותרת + רשימת קישורים (מייל / WhatsApp / Instagram).
7. **פוטר.**

### השתלטות הגרדיאנט (בתחתית)
כשמגיעים לאזור ה‑CTA/קשר: גרדיאנט בהיר (כחול→שמנת→כתום) **נפתח מפס דק במרכז ומתרחב**
עד שממלא את כל המסך (clip-path), **החץ מתפוגג**, וכל ה‑UI מתהפך לטקסט כהה (`is-takeover`).

## 7. אנימציות ואינטראקציות

- **גלילה חלקה** (Lenis) + **GSAP ScrollTrigger** לכל הכוריאוגרפיה.
- **חשיפת טקסט** על כל אלמנט, **וחוזרת** כשגוללים מעלה/מטה:
  - כותרות — כל שורה עולה ממסכה.
  - פסקאות — מילה‑מילה בגלים.
  - תוויות/קישורים/כרטיסים/פוטר — החלקה + הופעה מדורגת.
- **Hero intro** — המילים עולות אחרי שמסך הטעינה מתרומם.
- **חץ** — נשימה + הטיה + נקודת אור אחרי העכבר + נדידה בגלילה.
- **marquee** רץ, **טבעות מסלול** מסתובבות לאט, **כפתורים מגנטיים**, **סמן מותאם**.
- **כיבוד `prefers-reduced-motion`** — כל האנימציות נכבות.

## 8. ארכיטקטורה טכנית

- **Vite** (build) · **Vanilla JS** (ES modules, בלי framework).
- **GSAP + ScrollTrigger** · **Lenis** (smooth scroll).
- אין Three.js (היה, והוסר — ה‑bundle ירד מ‑701KB ל‑~140KB JS / gzip 52KB).
- האפקטים: **SVG + CSS + clip-path + mix-blend** בלבד (קל ומהיר).

### קבצים
```
index.html                  — מבנה + preloader + ה-mark (חץ נוזלי) + clipPath
vite.config.js              — base:'/Elevate/' ל-production
.github/workflows/deploy.yml— בנייה + פרסום אוטומטי ל-GitHub Pages
src/
  main.js                   — תזמור: scroll, cursor, theme toggle, magnetic,
                              אנימציות טקסט, אינטראקציית החץ, takeover, preloader
  styles/main.css           — כל העיצוב + שני הנושאים
  modules/
    smoothScroll.js         — Lenis + gsap.ticker
    preloader.js            — מסך טעינה (BUILD counter)
    cursor.js               — סמן מותאם
```
